Contoh Penggunaan API Upload ke CDN:

1. Upload dari Base64 (POST Request)

// JavaScript Fetch API const response = await fetch('https://your-api.com/api/upload', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ image: 'base64-encoded-image-data', filename: 'sms-pak-eko.png' }) }); const result = await response.json(); console.log(result);

2. Upload dari URL (POST Request)

// cURL Command curl -X POST "https://your-api.com/api/upload" \ -H "Content-Type: application/json" \ -d '{ "url": "https://example.com/sms-image.png" }'

3. Response JSON yang Dihasilkan:

{ "success": true, "data": { "cdnUrl": "https://uguu.se/abc123_sms.png", "filename": "sms-pak-eko.png", "size": 24567, "message": "Gambar berhasil diupload ke CDN" } }

4. Response Error:

{ "success": false, "error": "Gagal upload gambar ke CDN", "details": "Error message details" }
