Pratinjau Gambar:

Cara Menggunakan API untuk Gambar:

  1. Kustomisasi konten SMS di tab "Kustomisasi"
  2. Pilih format dan kualitas gambar di tab "API & URL"
  3. Generate URL API yang berisi parameter SMS Anda
  4. Gunakan URL tersebut untuk mengambil gambar SMS secara langsung
  5. Lihat pratinjau dan unduh gambar di tab "Pratinjau & Unduh"

Contoh Penggunaan API:

// Mengambil gambar SMS melalui API const apiUrl = 'http://example.com/api/sms-image?sender=Pak%20Eko&message=Test%20pesan&date=08/07/2025&time=14:55&format=png'; // Untuk menampilkan gambar langsung const img = document.createElement('img'); img.src = apiUrl; document.body.appendChild(img); // Atau untuk mengunduh gambar fetch(apiUrl) .then(response => response.blob()) .then(blob => { const url = window.URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = 'sms-pak-eko.png'; a.click(); });
